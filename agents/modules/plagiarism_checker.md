# Plagiarism Checker — 查重预警

## 触发条件

用户输入包含以下关键词时激活：
- "查重" / "检查抄袭" / "改写建议"
- 用户粘贴内容并问"这样写行不行"

## 工作流程

### Step 1: 文本比对

将用户输入的文本与 `data/` 目录下所有理论的定义部分进行文本相似度比较。

比较方法：关键词密度分析 + 关键句式匹配。

### Step 2: 相似度分级

| 级别 | 相似度 | 处理方式 |
|------|--------|---------|
| 🔴 高度相似 | >60% | 标记原文出处，提供 3 种改写方案，推荐正确引用格式 |
| 🟡 中度相似 | 30-60% | 提醒需要用自己的话重述，建议补充企业案例分析 |
| 🟢 低度相似 | <30% | 通过，无需处理 |

### Step 3: 输出报告

```markdown
## 查重预警报告

### 🔴 高度相似段落
检测到与理论库 [理论名称] 的原文高度相似：
> [匹配的原文片段]

**出处:** data/[类别文件].md → [理论名称]

**改写建议:**

方案 1（学术改写）：
[用自己的话重述，补充学术注释]
→ [例如] "根据[大师]在[年份]提出的[理论名称]，..."

方案 2（案例结合）：
[将理论定义嵌入具体企业案例]
→ [例如] "以[公司]为例，[理论名称]的[维度]体现在..."

方案 3（问题导向）：
[从管理问题出发引出理论]
→ [例如] "为什么[企业]面临[问题]？[大师]的[理论名称]提供了一个分析视角：..."

**正确引用格式:**
APA: [引用格式]
GB/T 7714: [引用格式]
```
