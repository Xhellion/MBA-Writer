# Theory Recommender — 理论推荐引擎

## 触发条件

用户输入包含以下关键词时激活：
- "推荐理论" / "用什么理论" / "理论推荐"
- "有什么理论可以用" / "适合什么理论"
- 在写作流程中处于"需要理论支撑"的环节

## 工作流程

### Step 1: 提取场景关键词

从用户描述的场景中提取 3-5 个关键词。
示例：
```
用户："我在写第三章，分析A集团的APP用户运营现状"
→ 提取: ["用户运营", "APP运营", "现状分析", "用户活跃度", "汽车行业"]
```

### Step 2: 理论匹配

在 `data/` 目录下的理论库中进行三层匹配：

| 匹配层 | 匹配字段 | 权重 | 说明 |
|--------|---------|------|------|
| 精确匹配 | keywords | ★★★★★ | 理论与场景关键词完全匹配 |
| 场景匹配 | scenarios | ★★★★☆ | 理论标注了适用场景 |
| 类别匹配 | category | ★★★☆☆ | 理论所在类别与场景相关 |

### Step 3: 排序与推荐

按综合得分排序，返回 Top 3-5 个理论，每个包含：

```markdown
**【推荐 N】理论名称**
- 适用理由: 为什么适合本章节
- 分析框架: 维度/步骤清单
- 关键文献: key_works
- 注意事项: 常见使用误区
```

### Step 4: 用户选择

用户选择某个理论后，可进一步：
- "生成分析框架" → 转 Template Generator
- "详细讲解" → 显示理论库全文
- "找案例" → 搜索该理论的企业应用案例

## 理论分类索引

### 战略管理
- 行业分析 → 波特五力、PEST、SWOT
- 竞争策略 → 三大通用战略、蓝海战略、安索夫矩阵
- 增长战略 → 颠覆性创新、企业生态系统

### 组织行为学
- 文化建设 → 霍夫斯坦德文化模型、阿德勒文化协调
- 组织变革 → 阿吉里斯不成熟-成熟理论、和谐管理
- 领导力 → 领导理论、管理技能模型

### 人力资源管理
- 员工激励 → 马斯洛需求层次、赫茨伯格双因素、X-Y理论
- 绩效考核 → 平衡计分卡、目标管理
- 人才发展 → 知识创造理论、学习型组织

### 运营管理
- 质量管理 → 六西格玛、全面质量管理
- 流程优化 → 价值链分析、服务蓝图
- 服务运营 → 服务利润链、服务运营管理理论

### 创新管理
- 技术创新 → 颠覆性创新、动态能力理论
- 商业模式 → 商业模式画布、价值共创
- 战略创新 → 蓝海战略、刺猬理念

### 其他
- 研究方法 → 案例研究方法、扎根理论
- 企业理论 → 委托代理、企业DNA、企业社会资本
