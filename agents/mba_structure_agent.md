# MBA Structure Architect — MBA 论文结构设计

> 继承 academic-paper Structure Architect 的全部能力。在 6 种通用结构之上增加 MBA 特有结构。

## MBA 论文结构模式

### 模式 7：MBA 学位论文 7 章结构

适用于 MBA 学位论文（最常用）：

| 章节 | 标题 | 建议字数 | 核心内容 |
|------|------|---------|---------|
| 第 1 章 | 绪论 | 3000-5000 | 研究背景、问题提出、研究意义、方法、技术路线 |
| 第 2 章 | 理论基础与文献综述 | 4000-6000 | 核心理论介绍、国内外文献回顾、研究缺口 |
| 第 3 章 | 行业/企业现状分析 | 4000-6000 | 行业背景、企业概况、现状描述、问题识别 |
| 第 4 章 | 问题诊断与成因分析 | 4000-6000 | 基于理论工具的问题分析、根因诊断 |
| 第 5 章 | 优化方案设计 | 5000-8000 | 基于理论指导的方案设计、实施路径 |
| 第 6 章 | 实施保障与预期效果 | 3000-5000 | 组织/资源/制度保障、效果评估 |
| 第 7 章 | 结论与展望 | 1000-2000 | 主要结论、研究局限、未来方向 |

### 模式 8：案例分析结构

适用于 MBA 案例分析论文：

1. 案例背景（企业概况、行业环境）
2. 问题描述（核心问题、表现症状）
3. 理论框架（选取 2-3 个相关理论）
4. 案例分析（用理论工具逐层解剖）
5. 对策建议（可操作的解决方案）
6. 结论与启示

### 模式 9：商业报告结构

适用于企业内部管理报告：

1. 执行摘要
2. 现状分析（数据+问题）
3. 理论依据
4. 方案设计
5. 实施计划

### 模式 10：战略分析报告结构

1. 外部环境分析（PEST + 五力）
2. 内部资源能力分析（价值链 + VRIO）
3. SWOT 综合分析
4. 战略定位与选择
5. 实施路径
6. 风险评估

## 结构选择逻辑

1. 论文类型 = "mba_thesis" → 默认 7 章结构
2. 论文类型 = "case_analysis" → 案例分析结构
3. 论文类型 = "business_report" → 商业报告结构
4. 其他 → 按 original 逻辑

## 理论映射

在结构设计完成后，自动为每个章节推荐相关理论：
- 第 2 章 → 所有理论库中的经典理论
- 第 3 章 → 行业分析类理论（PEST、五力等）
- 第 4 章 → 诊断分析类理论（SWOT、价值链等）
- 第 5 章 → 方案设计类理论（平衡计分卡、蓝海战略等）
