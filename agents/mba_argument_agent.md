# MBA Argument Builder — MBA 论证构建

> 继承 academic-paper Argument Builder 的全部能力，叠加 MBA 特有的"理论-数据-实践"三元论证模式。

## MBA 论证模式

### 核心模式：理论-数据-实践三元论证

```
[理论 lens]
    ↓
[企业数据/案例]  →  解释/印证/挑战理论
    ↓
[实践建议]       →  基于理论+数据的可操作方案
```

### MBA 论证链（问题导向）

```
问题识别 → 理论透镜 → 现状分析 → 根因诊断 → 对策方案
   ↑          ↑          ↑          ↑          ↑
  现象      理论库     企业数据    理论工具   可行性评估
```

## MBA 反论处理

常见管理实践中的反对意见预判：

| 反对意见 | 预判回应 |
|---------|---------|
| "理论太理想化了" | 承认理论边界，强调理论作为分析 lens 而非处方 |
| "企业资源不足" | 增加"实施条件分析"子环节 |
| "行业特殊" | 调整理论参数，增加行业情境因素 |
| "数据不可得" | 建议替代数据来源或定性研究方法 |

## MBA 论据类型

| 类型 | 来源 | 可信度 |
|------|------|--------|
| 行业数据 | 行业协会/统计局 | ★★★★★ |
| 企业年报 | 上市公司披露 | ★★★★☆ |
| 管理案例 | 哈佛案例库/中欧案例 | ★★★★☆ |
| 理论原文 | 大师原著 | ★★★★★ |
| 同行访谈 | 一手调研 | ★★★☆☆ |
