# MBA 论文评审标准

## 五维度评分框架

详见 `agents/mba_reviewer_agent.md` 的评审维度定义。

| 维度 | 权重 | 评分要点 |
|------|------|---------|
| 实践价值 | 30% | 对企业管理实践是否有指导意义，方案是否可落地 |
| 理论应用 | 25% | 理论选择是否恰当，使用是否准确深入 |
| 分析深度 | 20% | 数据分析是否充分，案例剖析是否透彻 |
| 逻辑结构 | 15% | 论文结构是否合理，论证是否严密 |
| 写作规范 | 10% | 格式是否规范，引用是否准确，语言是否学术化 |

## 常见扣分项

| 类型 | 典型问题 | 影响维度 |
|------|---------|---------|
| 选题过大 | "中国制造业转型升级研究"（MBA应聚焦具体企业） | 实践价值 |
| 理论不匹配 | 用波特五力分析企业内部管理问题 | 理论应用 |
| 理论研究法 | MBA用实证回归但不解释管理意义 | 实践价值 |
| 数据粗糙 | 二手数据为主，缺少一手调研 | 分析深度 |
| 有分析无结论 | 列出数据但不说管理启示 | 分析深度 |
| 方案不可操作 | "加强数字化转型"（怎么加强？） | 实践价值 |
| 逻辑跳跃 | 现象→结论，缺少理论+分析 | 逻辑结构 |
| 引用不规范 | APA/GB混用，缺少DOI | 写作规范 |
| 口语化严重 | "很多公司都在搞" | 写作规范 |
| 字数不达标 | MBA论文通常要求2-5万字 | 写作规范 |
| 查重率过高 | 直接复制百科定义 | 写作规范 |
